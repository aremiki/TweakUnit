# TweakUnit

**A lightweight and elegant addon to enhance World of Warcraft's default unit frames**

TweakUnit provides essential improvements to your player, target, and raid frames without the complexity of full-featured unit frame replacements. Perfect for players who love Blizzard's default UI but want that extra touch of customization.

## Features

### Player & Target Frames
- **Class Color Health Bars** - Display health bars in class colors for instant visual identification
- **Custom Textures** - Choose from a wide variety of statusbar textures including:
  - Default Blizzard textures
  - LibSharedMedia textures (if available)
  - Clean, modern alternatives to the default bars

### Raid Frames
- **Smart Fade System** - Automatically fades out-of-range raid members
  - Adjustable minimum and maximum opacity values
  - Helps you quickly identify who's in range for heals and buffs
  - Smooth, non-intrusive visual feedback

- **Advanced Font Customization**
  - **Name Text**: Customize font family, size, and outline
  - **Health Text**: Separate font settings for health values
  - **Color Control**: Choose your preferred color for health text
  - Full integration with LibSharedMedia fonts

- **Texture Options** - Apply custom statusbar textures to all raid frames

### User Interface
- **Intuitive Options Panel** - Clean, organized settings accessible via `/ri` command
- **Profile Support** - Full Ace3 profile management (multiple characters, easy switching)
- **Minimap Button** - Quick access icon (can be hidden if desired)
- **Live Updates** - All changes apply instantly, no UI reload required

## Installation

### Automatic (Recommended)
Install through your favorite addon manager:
- [CurseForge App](https://curseforge.overwolf.com/)
- [WoWUp](https://wowup.io/)

### Manual
1. Download the latest release
2. Extract the `TweakUnit` folder
3. Place it in your `World of Warcraft\_retail_\Interface\AddOns` directory
4. Restart World of Warcraft

## Usage

### Quick Start
1. Type `/ri` in-game to open the options panel
2. Navigate through the **Player**, **Target**, and **Raid** tabs
3. Adjust settings to your preference - changes apply immediately!

### Configuration Options

#### Player & Target
- Toggle class-colored health bars
- Select custom statusbar textures

#### Raid Frames
- **Fade Options**: Set minimum/maximum opacity for in/out of range members
- **Name Font**: Customize the appearance of player names
- **Health Font**: Customize health value display (font, size, outline, color)
- **Texture**: Apply custom textures to raid frame health bars

#### Profiles
- Create multiple profiles for different characters or situations
- Copy settings between profiles
- Reset to defaults anytime
